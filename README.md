# NVM Vanilla

## Usage

```
npm i -g nvm-vanilla
```

```
nvm current
nvm (which|where) <version>
nvm (list|ls)
nvm alias <name>

nvm install <version>
nvm uninstall <version>
nvm use <version>

nvm run <version> <args>
nvm exec <version> <command>

nvm alias <name> <version>
nvm unalias <name>
```

### For Windows

```
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned
```
